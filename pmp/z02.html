  • A：相关方登记册
    Related party register
  • B：工作说明书（SOW）
    Statement Of Work
  • C：团队沟通模板
    Team communication template
  • D：项目预测文件
    Project forecast file
正确答案：A 你的答案：
解析：说明解析：PMBOK（6）P512-13.1.3.1识别相关方-相关方登记册。包括相关方主要需求、期望等。选项B：SOW描述拟采购的产品、服务或成果，主要包括规格、所需数量、质量水平、绩效数据、履约期间、工作地点和其他要求等，不包括相关的期望。选项C：沟通模板记录相关方沟通的工具，与题目要求回答的内容不符。选项D：项目预测文件是项目监控文件，与题目要求回答的问题不符。
100、 在制定资源管理计划时，一位项目经理得知，完成一项关键任务所需的技能组合供不应求且需求很高。项目经理将这个问题记录在资源管理计划中。若要包含这个问题，应该更新哪一份其他文件？
While developing the resource management plan,a project manager learns that the skill not needed to complete a critical task is in short supply and high demanThe project manager notes this in the resource management plan.Which other file should I update to include this question?
  • A：采购管理计划
    Procurement management plan
  • B：风险管理计划
    Procurement management plan
  • C：培训手册
    Training manual
  • D：团队开发过程
    Team development process
正确答案：C 你的答案：
解析：定位：（理念）培训。 说明解析：所需技能供不应求且需求很高，就需要进行培训。选项A：采购管理计划是对采购的要求和规定，不包含培训内容和要求。 选项B：风险管理计划是对风险管理的规定，不包含培训内容和要求。选项D：团队开发过程与题目要求回答的内容不符。
